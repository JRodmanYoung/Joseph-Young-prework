About Me

 My name is Joe Young. I like to read books and bake bread. One of my favorite books is the second edition of Topology by James Munkres. It is a book about properties that are preserved under continuous transformations.

 My favorite kind of bread to make is probably whole wheat bread, because it is healthy and keeps me regular. Recently, I have experimented by making beer breads, which are breads that are made with beer. Unlike traditional yeast breads, which use yeast as the main leavening ingredient, beer breads use beer and baking powder as leavening.

 I like programming and learning new languages. I am currently teaching myself the language C++ using this site. It is tricky, but always worth the hard work! I don't enjoy coding in HTML and CSS as much as other languages, but Javascript is okay. I think Python is the best!
